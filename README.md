# README

QnA is clone of StackOverflow, written in Ruby On Rails. Used technologies:

1. Ruby 2.5
2. Rails 5.2.3
3. ActionStorage
4. ActionCable
5. Sphinx
6. Sidekiq and ActiveJob
7. REST API
8. OAuth
9. AJAX with ujs and jQuery.
10. Rspect and Capybara
11. Capistrano
10. Gems such as Devise, CanCanCan, Bootstrap and other
